📝 QuizMaster Java (Swing)
Un moteur de QCM léger et performant, développé intégralement en Java avec l'interface graphique Swing. Ce projet fonctionne exclusivement en local et ne nécessite aucune connexion internet ou base de données complexe.

🚀 Caractéristiques
Interface Graphique (GUI) : Utilisation native de javax.swing pour une expérience utilisateur fluide.

100% Local : Les questions sont chargées depuis des fichiers locaux (JSON, CSV ou TXT).

Zéro Dépendance : Pas de frameworks lourds ; uniquement les bibliothèques standards du JDK.

Système de Score : Calcul immédiat des résultats à la fin du questionnaire.
